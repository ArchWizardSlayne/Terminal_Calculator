# Terminal Calculator

A Python-based command-line calculator that performs basic arithmetic operations.

## Features

- Addition, subtraction, multiplication, and division
- Continuous calculations (chain operations)
- Input validation
- Error handling (division by zero protection)

## Usage

```bash
python main.py
```

Enter calculations in the format: `2 + 2` (space-separated)

After each calculation, you can continue operating on the result by entering just the operator and number, e.g., `+ 5`

Commands:
- `quit` - Exit the calculator
- `reset` - Start a new calculation

## Skills Demonstrated

- Python fundamentals
- Control flow (loops, conditionals)
- Input/output handling
- Error handling
- Clean, readable code structure
